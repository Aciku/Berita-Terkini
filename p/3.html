Apakah WiFi Bisa Melihat Riwayat Pencarian Anda?

Pertanyaan apakah WiFi dapat melihat apa yang kita buka mungkin membuat pengguna internet penasaran. Riwayat pencarian seringkali bersifat rahasia dan pribadi. Lantas, benarkah riwayat pencarian selama browsing dapat dilihat oleh pihak lain?


Jawabannya adalah ya, pemilik WiFi dapat melihat riwayat pencarian semua pengguna yang terhubung ke jaringannya. Ini disebabkan oleh kemampuan router modern untuk menyimpan catatan dari semua perangkat yang terkoneksi dengannya. Admin atau pemilik WiFi yang memiliki akses ke panel admin router dapat mengakses riwayat pencarian selama berselancar di internet.


Hal ini biasanya terjadi pada WiFi umum yang tersedia di hotel, sekolah, kantor, atau rumah orang lain. Selain itu, pemilik WiFi dan penyedia layanan internet (ISP) juga dapat mengatur siapa saja yang memiliki akses internet.


Berikut adalah beberapa tips untuk menyembunyikan riwayat pencarian Anda:


1. Gunakan Layanan VPN: VPN membantu melindungi riwayat pencarian internet dengan memodifikasi alamat IP dan mengenkripsi data internet.


2. Aktifkan Mode Penyamaran atau Incognito: Mode ini tidak menyimpan cookie, data, atau riwayat penelusuran saat browsing.


3. Gunakan Mesin Pencari yang Lebih Aman: Gunakan mesin pencari seperti DuckDuckGo atau Startpage yang tidak melacak atau menghubungkan riwayat pencarian dengan data pribadi.


4. Nonaktifkan Cookie pada Browser: Nonaktifkan cookie untuk mencegah situs web melacak kebiasaan dan preferensi pencarian Anda.


5. Sambungkan ke Server Proxy: Server proxy menyembunyikan alamat IP dari situs web yang Anda kunjungi.


6. Gunakan Browser yang Mendukung Privasi Pengguna: Gunakan browser seperti Tor Browser yang ramah privasi dengan mengumpulkan lebih sedikit data dan tidak menggunakan cookie.


Pilih salah satu tips di atas untuk menghalangi pihak lain melihat riwayat aktivitas internet Anda. Jika Anda tidak ingin pemilik WiFi atau ISP melihat riwayat pencarian Anda, disarankan untuk menggunakan koneksi internet pribadi.
